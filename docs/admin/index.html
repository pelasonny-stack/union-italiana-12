Conectar con GitHub

Para guardar cambios necesito un Personal Access Token de GitHub. Es una llave que solo este browser conoce y se usa para hacer commits al repo. NO se sube al sitio — queda solo en este navegador.

  1. Andá a https://github.com/settings/personal-access-tokens/new
  2. Token name: "Admin Unión Italiana 12"
  3. Expiration: 90 days (o lo que prefieras)
  4. Repository access: Only select repositories → seleccionar pelasonny-stack/union-italiana-12
  5. Permissions → Repository: Contents = Read and write; Metadata = Read-only (auto)
  6. Click Generate token. Copiá el token (empieza con github_pat_...) y pegalo abajo.
